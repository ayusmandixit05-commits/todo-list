# Todo-list

A simple and interactive **Todo List web application** built using **HTML, CSS, and JavaScript**.This app allows users to add tasks with a due date and time, display them in a structured layout,and delete tasks when they are completed. All todos are stored locally using**LocalStorage**, so the data persists even after refreshing the page.

# How it works
1.Enter a task name.
2.Select a due date and time.
3.Click **Add** to save the task.
4.Tasks are stored in LocalStorage and rendered dynamically.
5.Click **Done** to remove a task.
